# AGENTS.md

## Project
removebg-cli — CLI tool to remove image backgrounds using `rembg`.

## Commands
- Run: `uv run python removebgcli.py <args>` or `python removebgcli.py <args>`
- Lint/typecheck: none configured

## Workflow
- `removebgcli.py` is the main script
- `main.py` is boilerplate entry point
- Dependencies managed via `uv` with `pyproject.toml` and `uv.lock`

- Remote: `origin` → `github.com/itskreisler/removebg-cli`
- Branch: `main`
